# Extension Icons

These are placeholder SVG icons for the GBP Inspector extension.

To convert to PNG for Chrome Web Store:
1. Use an online SVG to PNG converter
2. Or use ImageMagick: `convert icon16.svg icon16.png`
3. Or use Figma/Illustrator to export as PNG

Icon sizes needed:
- icon16.png (16x16) - Small icon
- icon48.png (48x48) - Extension page
- icon128.png (128x128) - Chrome Web Store

The icons feature a location pin design with a gradient purple background (#667eea to #764ba2).
